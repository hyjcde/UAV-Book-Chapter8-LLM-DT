# 专著章节（不是会议短文）怎么写

对照已下载 OA 专著（Zhang Digital Twin Briefs；AI Structural Monitoring）的正文习惯：

1. **正向陈述为主**：先写本章做什么、读者学到什么。少用 “Rather than / 而不是 / ——” 靠对比撑句子。
2. **破折号少用**：Springer 英文多用逗号、分号、括号或短句。数字范围保留 `2--7`、`L1--L3`。
3. **章在全书中的位置一句说清**：Chapters 2–7 产出什么；本章消费什么；交给谁。
4. **目标可核对**：objectives / roadmap / outcomes（我们已有 `tab:ch8-outcomes`）。
5. **技术分层后接应用**：概念 → 契约/方法 → 案例走读 → 评价；少用小标题碎片。
6. **结尾 bridge**：指向下一节或第9章，不忽然另开话题。

本书第8章应按上述习惯维护。
