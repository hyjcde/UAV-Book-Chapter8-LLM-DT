# Chapter 8 writing policy（防查重 / 书章定位）

## 原则

1. **这是专著章节，不是论文改投。** 叙事服务全书 Phase I–IV，承接 Ch.4 重建、Ch.6 检测、Ch.7 GIS。
2. **本地 DefectGPT / IAP-RAG 稿件 = 文献之一。** 用第三方综述语气讲解「一类资产中心 RAG」，可引用，但禁止大段复述方法/实验原文。
3. **先讲别人做了什么，再讲本章综合出什么教学框架。** 课题组前期工作（张纪涵 GeoBIM-DT、赵本赟检测数据、杨贵东 Det-Recon-Reg、韩明桥工人监测 DT 等）与外部 LLM/Agent 工作并列。
4. **公式与术语可保留工程含义，表述必须改写。** 避免与 AUTCON 稿件连续相同句段。
5. **实验结果用「案例专栏」呈现，** 不写成论文 Results 节的克隆。

## 推荐叙事顺序（先广后专）

问题与全书接口 → **广泛的巡检/AEC 大模型应用版图** → **专业巡检约束（身份/出处/多模态/门控）** → 模型基础与接地栈 → 孪生知识封装（专业化） → 多模态检索 → Agent → 报告与运维 → 小结

不要一上来就讲某一条资产中心 RAG 管线；先让读者看到「大模型在 inspection 里一般怎么用」，再收束到本书决策层。
