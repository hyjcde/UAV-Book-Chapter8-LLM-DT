\chapter*{前言}
\markboth{前言}{前言}
\noindent
城市建筑存量老化与密集化并存，结构健康监测需要比传统人工吊绳巡检更安全、更快速、更一致的流程。无人机改变了立面影像采集方式，但文献中飞行规划、三维重建、缺陷识别与设施决策支持仍常彼此割裂。

本书\emph{AI-Driven UAV Building Inspection: An End-to-End Framework from Autonomous Flight to Digital Twins}（人工智能驱动的无人机建筑巡检：从自主飞行到数字孪生的端到端框架）按四阶段连续展开：智能数据采集（任务与运动规划）、数字表达与空间语境（三维重建与 GIS）、AI 驱动缺陷分析（数据集与检测模型），以及新一代决策支持。最后的技术章把数字孪生与大语言模型、视觉—语言模型融合，使读者能查询历史维修记录、生成巡检报告，并在可审计规则下获得预测性维护建议。

本书面向希望设计或部署自主巡检管线的研究生、研究者，以及土木、机器人与人工智能领域的实践者。

\chapter*{术语表}
\markboth{术语表}{术语表}
\noindent
Asset-ID 是本章对``资产标识''的简写：可维护宿主在设施台账与孪生中的稳定编码；语言答案与工单应点名同一编码。资产护照以资产为中心汇总近期观测、几何摘要与显式邻域。有界智能体可调用工具起草行动，但写入与复访须经人工与 geofence 门禁。引用或拒答要求答案中的 ID 属于检索证据包。模型 harness 是包裹可替换 LLM 的代码侧控制层。
